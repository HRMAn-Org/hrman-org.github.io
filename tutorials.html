[HOME](index.html) ### We are coming soon ![HRMAn](img/construction.svg "Coming soon") ###Installing KNIME and loading the HRMAn workflow KNIME is an open source data analytics, reporting and integration platform. KNIME integrates various components for machine learning and data mining through a modular data pipelining concept. A graphical user interface allows assembly of nodes for data preprocessing, for modeling and data analysis and visualization without, or with only minimal, programming. KNIME can be downloaded [__here__](https://www.knime.com/downloads). If you want to learn more about KNIME, you can check their [__homepage__](https://www.knime.com/). It also includes a [__forum__](https://www.knime.com/forum) which can be useful if you should encouter any issues with it. The video above shows you how to install KNIME and extensions needed to run HRMAn on your computer. After you have completed the steps you should be able to execute HRMAn and analyse your imaging data.
